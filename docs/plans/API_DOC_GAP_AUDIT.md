3. Parsed each route file for `router.get/post/put/patch/delete(...)` calls and detected `authenticateJWT`/`superAdminOnly` at router or route level.
4. Loaded `Servers/swagger.yaml` and indexed every operation by `METHOD /path` (OpenAPI path format, `/api` prefix removed).
5. Compared the two inventories.
