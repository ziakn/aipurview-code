# Decision: Persona Agent Rebrand

## Bağlam

AIPurview zaten 10 named domain agent'a sahip — risk, vendor, evidence, incident, model, policy, compliance, control-assessment, coordinator, base. Ama **isimlendirme generic** — rakipler (Vanta "Compliance Agent", Drata "Trust Agent", Diligent "AI Board Member") cesur, persona-bazlı isimler kullanıyor.

## Karar

Mevcut 10 agent'ı **rakip-uyumlu** isimlerle yeniden brand'le. **0 mühendislik** — sadece marketing.

## Önerilen Eşleşme

| Eski | Yeni Brand İsim |
|---|---|
| risk-agent | **Risk Triage Agent** |
| vendor-agent | **Vendor Due-Diligence Agent** |
| evidence-agent | **Evidence Collector Agent** |
| incident-agent | **Incident Response Agent** |
| model-agent | **Model Card Author Agent** |
| policy-agent | **Policy Drafter Agent** |
| compliance-agent | **Compliance Analyst Agent** |
| control-assessment-agent | **Control Test Agent** |
| coordinator | **Compass Coordinator** veya **Themis** (Yunan adalet) |
| base-agent | (silent — abstract base) |

## Yunan Mit Alternatifi (cesur)

- Themis (adalet) → Compliance Analyst
- Argus (100 göz) → Sentinel
- Mnemosyne (hafıza) → Memory Curator
- Cassandra (kehanet) → Predictive Risk
- Hephaestus (zanaatkâr) → Workflow Builder

## Sonuçlar

- Marketing wedge düşük maliyetle
- Substrate (10 agent) zaten mevcut
- Sadece UI label + dokümantasyon değişir
- Effort: 1-2 gün

## Sprint 1 Scope

Bu rebrand + 5 yeni agent (FRIA, DPIA, Drift Detective, Reg Horizon, AI Board Member) = 2 hafta'da 15 named agent.
